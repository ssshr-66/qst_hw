# 快速启动指南 - 仅前端演示

适用于只有 IDEA 的 Windows 电脑，快速查看前端效果（使用 Mock 数据，无需后端和数据库）

## 前置要求

确保已安装了 **Node.js**（版本 >= 16）

检查是否已安装：
```bash
node -v
npm -v
```

如果未安装，请从 [Node.js 官网](https://nodejs.org/) 下载安装。

## 启动步骤

### 1. 打开命令行

在项目根目录下，按住 `Shift` 键 + 右键点击空白处，选择"在此处打开 PowerShell 窗口"或"在此处打开命令提示符"

### 2. 进入前端目录

```bash
cd frontend
```

### 3. 安装依赖

```bash
npm install
```

这一步会下载所有前端依赖包，首次运行需要 1-2 分钟。

### 4. 启动开发服务器

```bash
npm run dev
```

看到类似以下输出表示启动成功：
```
VITE v5.x.x  ready in xxx ms

➜  Local:   http://localhost:5173/
```

### 5. 打开浏览器

在浏览器中访问：**http://localhost:5173/**
